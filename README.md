# orgB - T1

Utilizando o simulador Logisim e as três organizações do MIPS disponibilizadas (Monociclo, Multiciclo e Pipeline), implementar 5 instruções fornecidas para cada uma destas três versões. 

Intruções a serem implementadas:

## Instruções a serem implementadas: 
 #### JAL (Jump And Link)
 #### BLTZ (Branch On Less Than Zero)
 #### LD (Load Byte)
 #### MULT (Multiply)
 #### SLTI (Set On Less Than Immediate)
